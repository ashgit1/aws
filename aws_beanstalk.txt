Deploying a Web App Using Elastic Beanstalk:
--------------------------------------------

Link : http://docs.aws.amazon.com/gettingstarted/latest/deploy/overview.html

Elastic Beanstalk is a high-level deployment tool that helps you get an app from your desktop to the web in a matter of minutes. 
Elastic Beanstalk handles the details of your hosting environment—capacity provisioning, load balancing, scaling, and application health monitoring—so you don't have to.
Elastic Beanstalk supports apps developed in Java, PHP, .NET, Node.js, Python, and Ruby, as well as different container types for each language.
